We need answer the question asking for a computable countable structure
which is computably AUT-countable on a cone but has no finite parameter
set over which all automorphisms become $\Sigma^{in}_1$-definable from
the parameters and their images.

Construction used in answer.tex:
Let $V=[\omega]^{<\omega}$ be the countable $\mathbb F_2$-vector space.
Let $(D_i)_{i\in\omega}$ be a computable sequence of finite subsets of
$V$ in which every finite subset occurs cofinally often.  Structure has
sort $I=(\omega,<)$ and sort $V$; ternary relation
$R(i,x,y)$ iff $x+y\in D_i$.  Since each singleton $\{h\}$ occurs, every
translation $\tau_h:x\mapsto x+h$ is an automorphism, and these are all
automorphisms because $I$ is rigid and the singleton markers force
$f(x+h)=f(x)+h$.  In any copy, $\tau_h$ is computable by hardwiring the
copy's element $i$ with $D_i=\{h\}$ and searching for the unique
$R(i,x,y)$.

Nondefinability argument:
Given finite parameters $P$, let $M=\max(P\cap I)$ (or $-1$) and
$W=\operatorname{span}\bigcup_{i\le M}D_i$.  Choose nonzero
$h\notin W$ and $\pi=\tau_h$.  For $Q=P\cup\pi(P)$ choose $x$ so the
cosets $x+W$ and $x+h+W$ avoid $Q\cap V$.  For any existential formula
over $Q$ true of $(x,x+h)$, use a satisfying finite witness tuple.
Shift the entire finite part lying in the coset $x+h+W$ by a fresh
$t\ne0,h$, avoiding finitely many bad choices, preserving all literals
with low markers $i\le M$.  For high marker witnesses, replace them by
large markers whose $D_i$ are chosen to realize exactly the finite set of
new positive differences and avoid the negative ones; cofinally many
copies of each finite $D$ make this possible while preserving the marker
order pattern.  The same existential formula holds of $(x,x+h+t)$, not
in the graph of $\tau_h$.  Hence no existential disjunct can isolate the
graph pair, so the graph is not $\Sigma^{in}_1$ over $P\cup\pi(P)$.
